Descriptions of FFX Commands and Properties are generated during build time.

In addition, task-oriented Guides are available under `src/site/asciidoc/guides`:
- Preparing-a-Structure.adoc — End-to-end prep from PDB → solvation → minimization → NVT/NPT → production.
- Running-MD.adoc — Practical `Dynamics` usage, thermostats/barostats, I/O, restarts, and performance tips.
- Key-Files-and-Force-Fields.adoc — How `.key` files work, selecting AMOEBA parameter sets, mixing parameters.
- Constant-pH-MD.adoc — Setup and run `PhEnergy`/`PhMinimize`/`PhDynamics`, titration models.
- Alchemical-Free-Energy.adoc — Dual/quad topologies, lambda schedules, and analysis with `Thermodynamics`.
- OpenMM-GPU-Acceleration.adoc — Selecting devices, platforms, and precision; tuning performance.
- Troubleshooting-and-Reproducibility.adoc — Common issues, diagnostics, and reproducible workflows.
